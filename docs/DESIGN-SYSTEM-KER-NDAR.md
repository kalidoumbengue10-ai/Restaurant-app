## Tokens

- Page : `#F2E9DC`
- Surface : `#FBF7F0`
- Encre : `#1C1714`
- Terre : `#A33B20`
- Bissap : `#6E1723`
- Safran : `#D49A3A`
- Ligne : `rgba(28, 23, 20, .18)`
- Display : Iowan Old Style → Palatino → Georgia, 500–600
- Corps : Avenir Next → Segoe UI, 400–700
- Images : rayon 22 px
- Mini-cartes : rayon 15 px
- Panneau citation : rayon 28 px
- Boutons : capsule, disque action 30 px
- Motion : 160–280 ms, easing `cubic-bezier(.16, 1, .3, 1)`

## Mobile first

À 375 px, la navigation devient un panneau piloté par un bouton de 44 px, le menu passe en carrousel `84vw`, les comparatifs s'empilent et toutes les actions gardent une zone tactile minimale de 44 px.

## Accessibilité

Contrastes élevés, focus visible couleur safran, accordéon natif au clavier, un seul H1, textes courants de 16 px minimum sur mobile et réduction automatique des animations.
